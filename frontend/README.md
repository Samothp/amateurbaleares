# Frontend (Next.js)

Este directorio contiene la aplicación frontend del MVP.

Requisitos:
- Node.js 18+ y npm

Instalación y ejecución en local:

```bash
cd frontend
npm install
npm run dev
```

El servidor de desarrollo arranca en http://localhost:3000 por defecto.
